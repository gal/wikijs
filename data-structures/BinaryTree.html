Binary Trees

A tree is a connected undirected simple graph with no cycles.

A binary tree is a rooted tree in which:

  • Every node has at most 2 children.
  • The children of a node are ordered from left to right.

The depth of a binary tree is the length of the longest path from the root node to a leaf node.

Rooted Trees

We assume there is a root.
A rooted tree is a tree with one vertices designated the root, and all edges directed away from it.

Describing vertices in a rooted tree

If there is a directed edge from x to y, then x is the parent of y, and y is the child of x.
If y and w have the same parent, they are siblings of each other.
A vertex with no children is a leaf. A vertex with children is an internal vertex.
The ancestors of v are all the vertices from v to the root except for v.
The descendants of a vertex v are all the vertices that have v as an ancestor.

Binary Tree Node

  • element
  • leftchild
  • rightchild
  • optional: parent

Height of a node

The height of a node is the length of its longest directed path to a leaf.

height(node) = 0    # If node is a leaf
height(node) = 1 + max(height(left), height(right))
