The Set ADT

A set is a collection of elements, with no duplicates, and with no particular order among the elements.

They serve as the foundation for more common data types such as Maps (or dictionaries).

Implementation

Standard implementations include:

  • Array-based lists
  • Linked lists
  • AVL trees
  • Hash Maps
